Linux环境下编译gcc

# Linux环境下编译gcc

# 下载源码包

找一个比较快的镜像下载源码包

  1. https://gcc.gnu.org/mirrors.html
  2. ftp://ftp.mirrorservice.org/sites/sourceware.org/pub/gcc/releases/
$ wget ftp://ftp.mirrorservice.org/sites/sourceware.org/pub/gcc/releases/gcc-4.9.4/gcc-4.9.4.tar.bz2
1

# 解压

$ tar -xvzf gcc-4.9.4.tar.gz
$ cd gcc-4.9.4
1
2

# 下载依赖包

$ ./contrib/download_prerequisites
1

# 编译gcc

$ ./configure --prefix=/usr/local/gcc4.9.4 --disable-multilib
$ make -j 32
$ make install
1
2
3

执行，同时将环境变量加入 /etc/profile

$ export LD_LIBRARY_PATH=/usr/local/gcc4.9.4/lib:/usr/local/gcc4.9.4/lib64:$LD_LIBRARY_PATH
$ export PATH=/usr/local/gcc4.9.4/bin:$PATH
