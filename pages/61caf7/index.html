通过mongodb源码制作rpm安装包

# 通过mongodb源码制作rpm安装包

# 前言

使用rpmbuild制作mongodb安装rpm包，实现自定义安装、服务自启动、环境变量更新，加入yum源实现yum安装管理.

该rpm按以下约定完成安装：

  • 软件安装至/usr/local/mongo27017目录
  • 数据目录安装在/usr/local/mongo27017/data
  • 日志目录安装在/usr/local/mongo27017/logs
  • 服务属组为hadoop
  • 通过syctemctl实现服务自启动，服务名mongo27017.service
  • 端口27017
  • 配置文件中在/usr/local/mongo27017/mongo.conf

# 安装工具包

yum install rpmbuild rpmdevtools -y

# 初始化目录

rpmdev-setuptree rpmbuild

# 准备源码包

把原码包mongodb-4.0.12.tar.gz上传到SOURCES目录下，其他需要用到的包也放在这个目录下。

注：这里需要先解压源码包，在解压出来的目录下新建data、logs两个目录，然后再打包。

# 编写spec文件

在SPECS目录下添加mongodb.spec文件，文件内容如下，可以根据需要修改。

Name:       mongodb-test
Version:    4.0.12 
Release:    1.0.0
License:    GPL 
URL:        http://www.mongodb.com
Group:      applications/database 
Source:     mongodb-%{version}.tar.gz 
BuildRoot:  %(mktemp -ud %{_tmppath}/%{name}-%{version}-%{release}.el7.centos-XXXXXX) 
Packager:   weiyong@dba.com
Autoreq:    no
Summary: Mongodb For Test %{version} RPM

%description  
Mongodb %{version} Server. 

%define MONGO_PORT 27017
%define BASE_DIR /usr/local/mongo%{MONGO_PORT}
%define MONGO_USER hadoop
%define MONGO_GROUP hadoop

%prep 
%setup -q -n mongodb-%{version}
 
%build 

%install
rm -fr %{buildroot}
mkdir -p %{buildroot}%{BASE_DIR}
cp -r *  %{buildroot}%{BASE_DIR}

%pre
check_port() {
        netstat -nltup | grep -v "127.0.0.1" | grep $1 &>/dev/null
}
if [ -d %{BASE_DIR} ];then
        echo "Error: The base directory %{BASE_DIR} already exists, Aborting." ;
        exit 1
elif check_port %{MONGO_PORT}
then
        echo "Error: The port %{MONGO_PORT} already in used, Aborting." ;
        exit 1
fi

mkdir -p %{BASE_DIR}
#cp -r %{buildroot}%{BASE_DIR}/* %{BASE_DIR}/

cat > %{BASE_DIR}/mongo.conf <<EOF
port=%{MONGO_PORT}
dbpath=%{BASE_DIR}/data
logpath=%{BASE_DIR}/logs/mongo.log
pidfilepath=%{BASE_DIR}/mongo.pid
logappend=true
fork=true
directoryperdb=true
bind_ip_all=true
EOF

cat > %{BASE_DIR}/mongo.server <<EOF
#!/bin/sh

MONGOPORT=%{MONGO_PORT}
EXEC=%{BASE_DIR}/bin/mongod
CLIEXEC=%{BASE_DIR}/bin/mongo
PIDFILE=%{BASE_DIR}/mongo.pid
CONF=%{BASE_DIR}/mongo.conf

case "\$1" in
    start)
        if [ -f \$\$PIDFILE ]
        then
                echo "\$PIDFILE exists, process is already running or crashed"
        else
                echo "Starting mongod server..."
                \$EXEC -f \$CONF
        fi
        ;;
    stop)
        if [ ! -f \$PIDFILE ]
        then
                echo "\$PIDFILE does not exist, process is not running"
        else
                PID=\$(cat \$PIDFILE)
                echo "Stopping ..."
                \$CLIEXEC 127.0.0.1:\$MONGOPORT/admin --eval "db.shutdownServer({force:true,timeoutsec:10})"
                while [ -x /proc/\${PID} ]
                do
                    echo "Waiting for mongod to shutdown ..."
                    sleep 1
                done
                echo "Mongod stopped"
        fi
        ;;
    *)
        echo "Please use start or stop as first argument"
        ;;
esac
EOF

chown -R %{MONGO_USER}:%{MONGO_GROUP} %{BASE_DIR}
chmod 755 %{BASE_DIR}/mongo.server

%post

printf "[Unit]\nDescription=mongo%{MONGO_PORT}\n[Service]\nType=forking\nUser=%{MONGO_USER}\nGroup=%{MONGO_GROUP}\nExecStart=%{BASE_DIR}/mongo.server start\nExecStop=%{BASE_DIR}/mongo.server stop\nPrivateTmp=true\nTimeoutSec=60\n[Install]\nWantedBy=multi-user.target\n">/lib/systemd/system/mongo%{MONGO_PORT}.service
systemctl enable mongo%{MONGO_PORT}.service >/dev/null 2>&1
systemctl daemon-reload
systemctl start mongo%{MONGO_PORT}.service >/dev/null 2>&1
sed -i 's/\/usr\/local\/mongo27017\/bin://g' ~hadoop/.bash_profile
sed -i 's/\$PATH/\/usr\/local\/mongo27017\/bin:\$PATH/g' ~hadoop/.bash_profile
echo "Info: Mongodb %{version} install successed, run 'mongo --port %{MONGO_PORT} --host localhost' to connect!"

%preun
%{BASE_DIR}/mongo.server stop >/dev/null 2>&1

%postun 
rm -rf %{BASE_DIR}
rm -f /lib/systemd/system/mongo%{MONGO_PORT}.service
sed -i 's/\/usr\/local\/mongo27017\/bin://g' ~hadoop/.bash_profile

%clean
rm -rf %{buildroot}
 
%files 
%defattr(-, %{MONGO_USER}, %{MONGO_GROUP}) 
%attr(755, %{MONGO_USER}, %{MONGO_GROUP}) %{BASE_DIR}/*
 
%changelog

# 开始制作

rpmbuild -bb rpmbuild/SPECS/mongodb.spec

# 安装mongodb

执行rpm安装，安装完成后会直接启动mongodb：

# rpm -ivh mongodb-test-4.0.12-1.0.0.x86_64.rpm 
Preparing...                          ###################### [100%]
Updating / installing...
   1:mongodb-test-4.0.12-1.0.0        ###################### [100%]
Info: Mongodb 4.0.12 install successed, run 'mongo --port 27017 --host localhost' to connect!

## 卸载mongodb

不需要关闭服务直接卸载：


> rpm -e mongodb-test

>注意:
>
>在卸载服务时会把mongodb相关目录删除，如果需要请先备份数据；


## 编译命令说明

*rpmbuild* 

> -ba 既生成src.rpm又生成二进制rpm 
> 
> -bs 只生成src的rpm 
> 
> -bb 只生二进制的rpm 
> 
> -bp 执行到pre 
> 
> -bc 执行到 build段 
> 
> -bi 执行install段 
> 
> -bl 检测有文件没包含
> 
> 加入YUM源
> 
> createrepo --update 3rd/
