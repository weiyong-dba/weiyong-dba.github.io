使用Prometheus和Grafana监控Scylla数据库

# 使用Prometheus和Grafana监控Scylla数据库

# 前言

scylla-grafana-monitoring提供scylladb图形监控，整个安装过程需要联网，尤其是后面运行start-all.sh脚本时。如果是第一次运行，需要下载很多东西，耐心等待即可，下载地址。

# 系统需求

  • CPU - at least 2 physical cores/ 4vCPUs
  • Memory - 15GB+ DRAM
  • Disk - persistent disk storage is porportional to the number of cores and Prometheus retention period (see the following section)，eg. 6 * 16 * 200MB ~ 20GB (6node,16core per node,default 15 days retention time)
  • Network - 1GbE/10GbE preferred
  • Docker

# 安装依赖包

安装python3

tar -xzvf Python-3.6.8.tgz
./configure --prefix=/usr/local/python3 --with-ssl
make
make install
ln -s /usr/local/python3/bin/python3 /usr/bin/python3
ln -s /usr/local/python3/bin/pip3 /usr/bin/pip3
yum install -y git docker python-pip
pip3 install --upgrade pip
pip3 install pyyaml

# 通过git安装

clone 源码

git clone https://github.com/scylladb/scylla-grafana-monitoring.git

启动docker服务

service docker start

生成scylla_servers.yml文件

通过genconfig.py生成

./genconfig.py -d myconf -c cluster1 -dc dc1:192.168.0.1,192.168.0.2

copy prometheus配置文件

cd prometheus
cp scylla_manager_servers.example.yml scylla_manager_servers.yml
1
2

修改数据保存周期

--storage.tsdb.retention="60d"
1

启动服务 根据需要选择启动的版本-v指定版本。

./start-all.sh -s myconf/scylla_servers.yml -v 3.0,2.3
